## 发布流程指引

### 发布流程
```bash
# 版本号修改  "version": "0.2.31",
# 打包
pnpm run build
# 生成日志文件
pnpm run changelog
# 发布
pnpm run publish
```
另:   tag相关指令
```bash
#查看本地所有 tag
git tag

#查看远程所有 tag
git ls-remote --tags origin

# 打tag v1.0
git tag v1.0

# 推送tag v1.0到远程
git push origin v1.0

# 删除本地tag v1.0
git tag --delete v1.0

#删除远程tag v1.0
git push origin --delete  v1.0
```
